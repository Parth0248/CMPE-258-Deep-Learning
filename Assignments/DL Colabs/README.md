# Deep Learning Colab Notebooks

CMPE-258 Deep Learning course assignment notebooks covering mathematical foundations, neural network implementations from scratch, and framework-specific tutorials.

## Math Foundations

| Notebook | Description |
|----------|-------------|
| `linear_algebra_for_deep_learning` | Vectors, matrices, and operations essential for deep learning |
| `calculus_for_deep_learning` | Derivatives, chain rule, gradient descent, and backpropagation |
| `probability_for_deep_learning` | Probabilistic concepts, distributions, likelihood, and entropy |
| `probability_fundamentals_for_deep_learning` | Core probability theory and statistical foundations |
| `numpy_foundations_for_deep_learning` | NumPy array operations for numerical computing |

## Neural Networks from Scratch

| Notebook | Description |
|----------|-------------|
| `Colab 1 - neural_networks_from_scratch` | End-to-end neural networks using only NumPy (XOR, moons, spirals, MNIST) |
| `why_deep_learning_works_geometric_intuition` | Geometric intuition behind why deep learning works |
